# 🎱 Roller Madness

A Unity WebGL rolling game developed as part of the **Introduction to Game Development** course at **Michigan State University**.

The player controls a ball navigating through open environments and obstacles, focusing on rigidbody physics, camera follow mechanics, and level design fundamentals.

---

## 🚀 Live Demo

👉 **[Play in Browser →](https://unity-roller-madness.tremok.com)**

---

## 🎮 Gameplay

- Roll the ball through challenging environments
- Avoid obstacles and stay on course
- Physics-driven movement using Unity's Rigidbody system

---

## 🛠️ Tech Stack

- **Engine:** Unity (WebGL export)
- **Language:** C#
- **Hosting:** [Vercel](https://vercel.com)
